# 🚀 Disnord Chat
Ein moderner, leichtgewichtiger Chat im Discord-Stil.

## Features
- 💬 Echtzeit-Chat mit Socket.io
- 🎨 Modernes Light-Design (Poppins Font)
- 👥 Aktive Nutzerliste
- 🛠️ Server-Erstellung und Profil-Einstellungen

## Installation
1. Repository klonen
2. `npm install` ausführen
3. `npm start` eingeben
